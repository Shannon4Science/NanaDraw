## 页面

| 路由 | 组件 | 权限 | 说明 |
|------|------|------|------|
| `/` | LandingPage | 否 | 落地页 — 头图、功能卡片、「开始绘图」主按钮 |
| `/draw` | DrawPage | 是 | 主编辑器 — draw.io 画布 + AI 面板 |
| `/draw?canvas=ppt` | DrawPage | 是 | PPT 编辑器变体 |
| `/draw?project={id}` | DrawPage | 是 | 已保存项目编辑器（持久化 URL） |
| `/projects` | ProjectsPage | 是 | 项目列表 — 带缩略图的网格 |
| `/admin` | AdminDashboard | 管理员 | 仪表盘 — 统计、积分流水 |
